Hướng Dẫn Sử Dụng

Bước 1: Chọn Template

Nhấp vào nút "Xem Live" hoặc "Code" để xem chi tiết template bạn muốn sử dụng

Bước 2: Copy Code

Copy toàn bộ HTML, CSS, JS từ template vào project của bạn

Bước 3: Tùy Chỉnh

Thay đổi màu sắc, text, hình ảnh để phù hợp với brand của bạn

Bước 4: Triển Khai

Cộng tác không gặp lỗi, websiet của bạn đã sẵn sàng hoạt động!

Ví Dụ Custom CSS Variables:

:root { --primary-color: #0066cc; --secondary-color: #004499; --accent-color: #ff6b6b; --primary: #6366f1; --secondary: #4f46e5; --accent: #ec4899; } /* Chỉ cần thay đổi giá trị màu ở đây để thay đổi toàn bộ giao diện */
